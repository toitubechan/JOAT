## Card: What it is
image: psychMemoryTypesHero
Memory isn't one single thing — there are different **types**, each working differently. The main split is between **short-term** (or working) memory, which holds a little for a brief time, and **long-term** memory, which stores vast amounts more durably. Knowing the types clarifies how remembering actually works.

## Card: Why it matters
Understanding memory types explains everyday experiences — why you can hold a phone number just long enough to dial it but forget it seconds later, and why some things stick for life. It also informs how to learn effectively, by understanding how information moves from fleeting to lasting memory.

## Card: Key terms
- **Short-term / working memory**: holds a little, briefly.
- **Long-term memory**: stores a lot, durably.
- **Consolidation**: moving memories into long-term storage.
- **Memory capacity**: how much a memory type can hold.

## Card: Short-term vs. long-term
image: psychMemoryTypesShortTermVsLongTermHero
**Short-term (working) memory** holds a small amount of information for a short time — like keeping a phone number in mind to dial it. Its **capacity** is famously limited (only a handful of items at once). **Long-term memory** stores vast amounts for days to a lifetime. The journey from short-term to long-term is how something fleeting becomes a lasting memory.

## Card: How memories become lasting
For information to stick, it must move from short-term to long-term storage through **consolidation** — strengthened by attention, repetition, meaning, and (importantly) sleep. This is why cramming briefly held facts often fails, while spaced, meaningful practice creates lasting memories. There are also subtypes of long-term memory, like memories of *facts* versus *skills* (you don't recall learning to walk, but you can do it).

## Card: Common mistakes
Common misunderstandings: thinking memory is one single system; overestimating short-term capacity (it's quite limited); assuming information automatically becomes long-term (it needs consolidation); and not realizing why repetition, meaning, and sleep matter for making memories stick. Knowing the types explains both fleeting and lasting memory.

## Card: Quick recap
Memory comes in types: short-term (working) memory holds a little briefly and has limited capacity, while long-term memory stores vast amounts durably. Information becomes lasting through consolidation, aided by attention, repetition, meaning, and sleep. There are also subtypes, like memory for facts versus skills. Knowing the types clarifies how memory works.
