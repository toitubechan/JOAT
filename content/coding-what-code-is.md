## Card: What it is
image: codingWhatCodeHero
**Code** is a set of precise instructions written in a language a computer can understand and follow. Programming is the act of writing these instructions to make a computer do something. At its heart, code is just step-by-step directions — like a very precise recipe — that tells the computer exactly what to do.

## Card: Why it matters
Code runs the digital world — every app, website, and device. Understanding what code actually *is* demystifies technology, opens the door to learning programming, and helps you grasp how the software around you works. It's increasingly valuable literacy, and the core idea is far simpler than it seems.

## Card: Key terms
- **Code**: precise instructions for a computer.
- **Programming**: writing code.
- **Programming language**: a language for writing code.
- **Syntax**: the exact rules for writing code correctly.

## Card: Instructions, like a recipe
image: instructionsRecipeHero
**Code** is essentially a list of precise instructions — like a recipe telling a cook exactly what to do, step by step. The computer follows these instructions exactly and literally. It can't guess your intent or fill in gaps; it does precisely what the code says. This is why code must be exact: a tiny mistake can cause the whole thing to fail, because the computer follows instructions without judgment.

## Card: Languages and syntax
We write code in **programming languages** — special languages designed to be precise and unambiguous (unlike human languages, which are full of nuance). Each language has its own **syntax** (exact rules for how to write it correctly), like grammar. There are many languages for different purposes, but they all share the same core idea: giving the computer clear, precise, step-by-step instructions it can follow.

## Card: Common mistakes
Common misunderstandings: thinking code is mysterious or beyond ordinary people (it's just precise instructions, learnable by anyone); imagining computers understand intent (they follow code literally and exactly); thinking you must master complex math first (basic logic matters more); and being intimidated by syntax. Code is simply step-by-step directions written in a precise language.

## Card: Quick recap
Code is precise, step-by-step instructions written in a programming language that a computer follows exactly and literally — like a very exact recipe. Because computers follow instructions without judgment, code must be precise; small mistakes cause failures. Different languages have different syntax but share this core idea. Code isn't mysterious — it's learnable instructions anyone can begin to understand.
