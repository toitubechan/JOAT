## Card: What it is
image: psychMemoryMythsHero
Memory is surrounded by popular **myths** — from "photographic memory" to "we only use 10% of our brain." This card set debunks the common ones, replacing misconceptions with what's actually true. Knowing the myths helps you treat your memory (and claims about it) more realistically.

## Card: Why it matters
Memory myths lead to false expectations, wasted money on "brain boosters," and misplaced trust in unreliable memories. Clearing them up gives you an accurate understanding of your own mind, helps you spot pseudoscience, and sets realistic expectations for what memory can and can't do. Truth beats myth for using your memory well.

## Card: Key terms
- **Myth**: a popular but false belief.
- **Photographic memory**: the disputed idea of perfect recall.
- **Repressed memory**: a contested idea about hidden trauma memories.
- **Memory as recording**: the false idea memory replays perfectly.

## Card: Common memory myths busted
image: psychMemoryMythsCommonMemoryMythsBustedHero
- "**Photographic memory**" (perfect, exact recall) — there's little solid evidence true photographic memory exists; even people with excellent memories don't recall *perfectly*.
- "Memory works like a **recording**" — it's reconstructed and fallible, not a faithful replay.
- "We use only **10% of our brain**" — a complete myth; we use virtually all of it.
- "**Brain-training games** dramatically boost memory" — they mostly just make you better at those games.

## Card: What's actually true
The reality: memory is reconstructive and fallible (even confident, vivid memories can be wrong); it can be improved with technique and healthy habits, but not turned "photographic"; and the brain is used fully, not at 10%. Eyewitness memory and "recovered" memories can be unreliable, which has real consequences. Realistic understanding beats the comforting but false myths.

## Card: Common mistakes
Common mistakes: believing some people have perfect photographic recall; trusting memory as an exact recording (it isn't); spending on "brain-training" promising big memory gains; and assuming you only use 10% of your brain. The truth — memory is fallible and reconstructive but improvable with real techniques — is more useful than any myth.

## Card: Quick recap
Memory myths abound: "photographic memory" lacks solid evidence, memory isn't a perfect recording (it's reconstructed and fallible), the "10% of our brain" claim is false, and brain-training games mostly improve only those games. The truth — memory is fallible but improvable with genuine techniques — is more useful and accurate than the comforting myths.
