## Card: What it is
image: periodicTableHero
The **periodic table** is the famous chart listing all the known **elements** — the basic types of atom. Far from a random grid, it's cleverly organized so that an element's position tells you about its properties. It's chemistry's master map, organizing the building blocks of everything.

## Card: Why it matters
The periodic table is one of science's greatest achievements, packing huge amounts of information into one chart. Understanding even its basic logic demystifies the intimidating grid on the classroom wall and reveals why elements behave as they do. It turns a wall of symbols into an ordered, sensible system.

## Card: Key terms
- **Element**: a substance made of a single type of atom.
- **Periodic table**: the organized chart of all elements.
- **Symbol**: a one- or two-letter abbreviation (like O for oxygen).
- **Group**: a column of elements with similar properties.

## Card: An ordered catalog
Each box in the table is one **element**, shown by its **symbol** (like H for hydrogen, O for oxygen, Fe for iron). The elements are arranged by their number of protons, increasing across the table. This isn't arbitrary — the order reflects atomic structure, which is why the table is so powerful: position encodes properties.

## Card: Patterns reveal properties
image: periodicPatterns
The genius is the patterns. Elements in the same **group** (vertical column) have similar properties — for example, one column holds reactive metals, another holds unreactive gases. Properties repeat in a regular ("periodic") way as you move across, which is where the name comes from. Knowing where an element sits tells you a lot about how it behaves.

## Card: Common mistakes
Common misunderstandings: seeing the table as a random list to memorize rather than a meaningful order; not realizing position reveals properties; and being intimidated by it, when the basic logic is simple. You don't need to memorize it — just understand that it's an *organized* map where location carries meaning.

## Card: Quick recap
The periodic table is the organized chart of all elements (the basic types of atom), each shown by a symbol. Elements are arranged by proton number, and the layout reveals patterns: those in the same column share similar properties. It's not a random list to memorize but a meaningful map where position encodes behavior.
