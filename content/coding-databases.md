## Card: What it is
image: databasesHero
A **database** is an organized collection of information stored so it can be easily searched, retrieved, and updated. Almost every app and website uses one behind the scenes — to store users, posts, products, orders, and more. Databases are how software remembers and manages large amounts of data reliably.

## Card: Why it matters
Databases power virtually all the software you use — every login, every saved post, every online order relies on one. Understanding the basic idea demystifies how apps store and find your information, and it's foundational for understanding software and back-end development. It explains how programs handle data at scale.

## Card: Key terms
- **Database**: an organized store of information.
- **Records**: individual entries (like one user or one order).
- **Query**: a request to find or change data.
- **Structured data**: data organized into a defined format.

## Card: Organized so it's findable
A **database** is essentially a highly organized store of information — often imagined like tables (rows and columns) of **structured data**. Each **record** is an entry (like one user, with their name, email, and so on). The key is organization: data is structured so the software can quickly find exactly what it needs among millions of records, rather than searching messily. Good organization is what makes a database powerful.

## Card: Querying data
image: queryingDataHero
Software interacts with a database through **queries** — requests to find, add, change, or delete data. For example, "find the user with this email," or "add this new order," or "show all products under this price." The database handles these requests efficiently, even across huge amounts of data. This is how an app instantly pulls up your account or shows search results: it queries its database behind the scenes.

## Card: Common mistakes
Common misunderstandings: thinking data is just stored randomly in files (databases organize it for fast, reliable retrieval); not realizing nearly every app uses one; imagining databases are only giant or complex (they range from simple to huge); and not grasping that queries are how software finds and changes data. A database is an organized, searchable store that lets software manage information at scale.

## Card: Quick recap
A database is an organized collection of information, often structured like tables of records, stored so software can quickly search, retrieve, and update it. Apps interact with databases through queries — requests to find or change data. Nearly every app and website uses one behind the scenes for users, posts, products, and orders. Databases are how software manages large amounts of data reliably.
