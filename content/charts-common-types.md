## Card: What it is
image: dataChartsCommonTypesHero
There are a few **common chart types** — bar charts, line graphs, pie charts, and others — each suited to showing different kinds of data. Knowing which is which, and what each does well, helps you read them correctly and recognize when the wrong type is being used. It's basic chart literacy.

## Card: Why it matters
Each chart type tells a particular kind of story, and using the wrong one (deliberately or not) can confuse or mislead. Recognizing the main types and their purposes helps you interpret data accurately and judge whether a chart fits its data. It's foundational for reading the charts you encounter daily.

## Card: Key terms
- **Bar chart**: bars comparing quantities across categories.
- **Line graph**: a line showing change over time.
- **Pie chart**: slices showing parts of a whole.
- **Scatter plot**: dots showing the relationship between two things.

## Card: Bars, lines, and pies
image: dataChartsCommonTypesBarsLinesAndPiesHero
- **Bar charts** compare amounts across categories (sales by product) — good for comparisons.
- **Line graphs** show how something changes over time (temperature across a year) — good for trends.
- **Pie charts** show parts of a whole (budget split into categories) — good for proportions, but only when the parts add up to a meaningful 100%.
Matching the chart to the data is key.

## Card: Other useful types
Beyond the basics: **scatter plots** use dots to show the relationship between two variables (is there a pattern between two things?). There are many more (histograms, area charts, maps), but bar, line, pie, and scatter cover most everyday charts. The key skill is recognizing what kind of story each is built to tell — comparison, trend, proportion, or relationship.

## Card: Common mistakes
Common misunderstandings: not knowing which chart type suits which data; pie charts used when slices don't form a meaningful whole (or there are too many slices to read); line graphs used for non-time data where they imply a false trend; and being confused by an inappropriate chart type. Recognize the type and ask whether it fits the data.

## Card: Quick recap
Common chart types each tell a particular story: bar charts compare categories, line graphs show change over time, pie charts show parts of a whole, and scatter plots show relationships between two things. Matching the chart to the data matters — the wrong type confuses or misleads. Recognizing each type and its purpose is basic chart literacy.
