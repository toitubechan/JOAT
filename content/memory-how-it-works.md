## Card: What it is
image: psychMemoryHowItWorksHero
**Memory** is how your brain stores information and retrieves it later. It's not like a video recorder that captures and replays events perfectly. Instead, memory works in stages — taking information in, storing it, and reconstructing it on recall. Understanding this changes how you think about remembering and forgetting.

## Card: Why it matters
Memory underlies learning, identity, and daily functioning. Understanding how it actually works helps you learn more effectively, trust your memory appropriately (not too much), and make sense of why you forget. It also dispels common myths. It's foundational knowledge about your own mind.

## Card: Key terms
- **Encoding**: getting information into memory.
- **Storage**: holding information over time.
- **Retrieval**: recalling stored information.
- **Reconstruction**: rebuilding a memory rather than replaying it.

## Card: The three stages
image: psychMemoryHowItWorksTheThreeStagesHero
Memory works in three stages: **encoding** (taking information in and forming a memory), **storage** (holding it over time), and **retrieval** (bringing it back when needed). Problems can occur at any stage — you might never encode something well (so it never stuck), or fail to retrieve something that's actually stored. This framework explains a lot about remembering and forgetting.

## Card: Memory is reconstructed
A crucial truth: memory is **reconstructed**, not replayed. When you recall something, your brain rebuilds it from pieces — and fills gaps, often inaccurately. So memories can feel vivid and certain yet be wrong, and they can change over time. Memory isn't a faithful recording; it's an active, fallible reconstruction. This explains why eyewitnesses and our own confident memories can be mistaken.

## Card: Common mistakes
Common misunderstandings: thinking memory is like a video recording (it's reconstructed and fallible); believing a vivid memory must be accurate (vividness isn't accuracy); not realizing forgetting can be an encoding failure (you never stored it well); and over-trusting your memory. Understanding memory's reconstructive nature brings useful humility.

## Card: Quick recap
Memory works in three stages — encoding (taking in), storage (holding), and retrieval (recalling) — and problems can occur at any. Crucially, memory is reconstructed, not replayed: the brain rebuilds memories and fills gaps, so even vivid memories can be wrong. It's an active, fallible process, not a faithful recording.
